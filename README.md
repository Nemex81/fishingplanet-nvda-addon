# FishingPlanet NVDA Addon

Addon NVDA per il gioco FishingPlanet con sistema OCR intelligente a zone predefinite.

## Features

- ✅ Sistema OCR automatico basato su Windows OCR
- ✅ 4 zone predefinite ottimizzate (pesca, negozio, centro, completo)
- ✅ Filtro intelligente duplicati
- ✅ Comandi rapidi per cambio zona
- ✅ Feedback audio per stati OCR
- ✅ Ottimizzato per performance e stabilità

## Quick Start

1. Installare l'addon `.nvda-addon`
2. Avviare FishingPlanet
3. Premere `NVDA+ALT+2` (zona pesca)
4. Premere `NVDA+ALT+L` (avvia OCR)

## Building from Source

1. Clone repository: `git clone https://github.com/Nemex81/fishingplanet-nvda-addon.git`
2. Zip contenuto cartella `addon/`
3. Rinominare file `.zip` in `.nvda-addon`

## Requirements

- NVDA 2021.1+
- Windows 10/11 with OCR language pack
- FishingPlanet game
